Panels

Consists of a drawer and a main container.

It is assumed that there will be a toolbar at the main content.

USE

panel|[nodim|alwayshide|fullheight]

+ drawer

+ main

nodim — Drawer doesn't dim the toolbar(fixed) when opened.

alwayshide — Always hides the drawer even at big screens.

fullheight — Takes up the full height of the screen. This is useful if you don't have a toolbar or is on scroll mode.

When having a toolbar at the main element, the content should have a content attribute.

panel-target={id} — to be attached onto an element to toggle the drawer, targeting the panel id.

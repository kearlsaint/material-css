Helpers

Material CSS comes with various helpers to ease out complications, but don't use too much as it will result to complications. Wait what?

fluid — adds display: block and width: 100% to the element.
italic — italicizes the text inside the element.
oblique — same as above. For the difference, read here.
bold — bolds the text inside the element.
lighter — changes the font-weight to 300 if available.
thin — changes the font-weight to 100 if available.
muted — changes the text color to #eee.
smaller — changes the font weight to smaller.
align-left — left-aligns the text.
align-right — right-aligns the text.
align-center|centered — center-aligns the text.
align-justify|justified — justifies the text.
v-{top|middle|bottom} — vertical-alignment.
uppercase|lowercase|capitalize — text-transforms.
padded|nopadding — adds an 8px padding; removes padding.

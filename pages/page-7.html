Inputs

Pretty basic inputs and styled textareas.

You can add a code attribute to a textarea field for displaying codes.

Checkboxes and radios are only in black.

USE

TEXTAREA
[vertical|horizontal]|noresize

CHECKBOXES/RADIOS — Must be associated with a label.

Forms

Forms can be vertical and horizontal

USE

vertical|horizontal defaults to vertical
labels and inputs should be inside a control attributed element
controls should be a direct child of a form.

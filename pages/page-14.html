Pages

A simple page function. No animations.

USE

pages

+ page|page-id={id}|page-title={title}|default

page-target={id}

default — The default page to be opened when the site is loaded.

page-id={id} — To be used with an element with page-target attribute for it to show.

page-title={title} — The title of the page; used to change the text inside toolbar > title.

page-target={id} — Element to toggle the page with the same id. You get the idea.
